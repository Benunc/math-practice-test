# CP Statistical Modeling Practice Test

An interactive HTML practice test for CP Statistical Modeling Final Exam preparation.

## Features

- **Progress Tracking**: Real-time progress bar showing completion percentage
- **Scoreboard**: Live tracking of correct, wrong, and unanswered questions
- **Interactive Questions**: Multiple choice, select-all-that-apply, and text input questions
- **Answer Checking**: Instant feedback with color-coded correct/incorrect answers
- **Results Summary**: Final score percentage and detailed results

## Usage

Simply open `math_practice_test.html` in any modern web browser. No server or installation required - it's a standalone HTML file with embedded CSS and JavaScript.

## Test Content

The practice test covers:

- **Part I**: Exploring and Understanding Data (8 questions)
- **Part II**: Exploring Relationships between Variables (4 questions)
- **Part III**: Gathering Data (6 questions)
- **Part V**: From the Data at Hand to the World at Large (6 questions)

## Topics Covered

- Categorical vs. quantitative data
- Normal distributions and z-scores
- Outliers and boxplots
- Scatterplots and correlation
- Linear regression
- Sampling methods and bias
- Confidence intervals
- Hypothesis testing
- Central Limit Theorem
